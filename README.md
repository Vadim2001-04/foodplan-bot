# FoodPlan AI - Telegram Bot

Telegram-бот для планирования питания и автоматизации заказа продуктов.

## Функционал

* Регистрация пользователя (диета, бюджет)
* Генерация меню на 7 дней
* Формирование списка покупок (без дублей)
* Интеграция с API доставки

## Установка

1. Клонируйте репозиторий:
```bash
git clone <your-repo-url>
cd foodplan-bot
```
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Настройте токен:
```bash
BOT_TOKEN = 'your_token_here'
```

4. Запустите бота:
```bash
python main.py
```
